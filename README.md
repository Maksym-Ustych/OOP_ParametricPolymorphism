# OOP Parametric Polymorphism (C++)

## Опис
Реалізація параметричного поліморфізму з використанням базового класу `Chislo` та похідного класу `Matrix`.

## Мета
Ознайомлення з віртуальними функціями та вказівниками на базовий клас.

## Суть
Програма обчислює факторіали елементів масиву за допомогою віртуальної функції базового класу.

## Технології
- C++
- Visual Studio
